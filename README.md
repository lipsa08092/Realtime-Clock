## Realtime Clock
A simple web application that shows the current time in real-time.

- This Realtime Clock app displays the current time with hours, minutes, and seconds.
- It updates every second and has a clean, responsive design suitable for learning DOM manipulation and JavaScript.

## Features
- Displays hours, minutes, and seconds
- Optional: 12-hour or 24-hour format
- Fully responsive design
- Updates every second in real-time

## Tech stack
HTML • CSS • JavaScript
